# 카카오톡 순위 측정기 카카오-카운터(kakao-counter)

### 특징
- 카카오톡 단톡방, 오픈채팅방 등의 대화목록을 분석하여 지정된 기간에서 대화수가 가장 많은 순서대로 명단을 작성합니다.
- 허수를 예방하기 위해 장타, 단타는 따지지 않으며 글자수를 기반으로 합니다. 이때 영문은 1 byte, 한글과 같은 동아시아 문자는 2 byte라는 차이 또한 감안합니다.
- 반복적인 초성 입력(ㅋㅋㅋ, ㅎㅎ, ㄷㄷ 등)은 글자수에 모두 반영되지 않으며, 연속된 글자 중 최대 3자까지 카운트 합니다.

### 빌드 없이 실행하기
- 컴퓨터에서 파이썬을 설치합니다.
- 터미널에서 'python kakao_counter.py'를 입력하여 실행합니다.
- 또는 확장자 연결을 통해 파이썬으로 직접 실행하도록 설정하실 수 있습니다.
